# Employee Salary Prediction

## Overview

This project predicts an employee's **salary** based on **experience** and **education level** using a **Linear Regression** machine learning model.

## Objective

* Build a regression model to estimate employee salary.
* Understand how experience and education affect salary.

## Technologies

* Python
* Pandas
* Matplotlib
* Scikit-learn

## Dataset

Columns used:

* **Experience** – Years of work experience
* **Education** – Education level
* **Salary** – Employee salary

## How to Run

```bash
pip install pandas matplotlib scikit-learn
python salary_prediction.py
```

## Model

The project uses **Linear Regression** to train and predict employee salary.
